# 网络安全日志分析平台交付里程碑

## M1. 基础工程与本地运行
- 目标：在 Windows 本地建立可启动的前后端分离工程骨架。
- 交付物：
  - 前端 Vite + React + TypeScript 工程
  - 后端 FastAPI 工程
  - SQLite 初始化与基础配置
  - Windows 启动脚本
  - 根目录使用说明
- 验收标准：
  - 前端可启动并访问
  - 后端可启动并返回健康检查
  - SQLite 数据库文件可自动创建
  - 本地配置可通过 `.env` 管理

## M2. 项目与数据导入闭环
- 目标：完成项目管理、文件导入、基础字段探测与数据版本登记。
- 交付物：
  - 项目创建、查询接口与页面
  - `.log`、`.csv`、`.xlsx` 文件上传接口
  - 文件元数据记录
  - 字段探测与映射确认接口
  - 标准化数据版本落盘为 Parquet
- 验收标准：
  - 能创建项目并绑定导入文件
  - 能识别文件类型并返回字段预览
  - 能生成 `DatasetVersion`
  - 数据版本可追踪原始文件与 schema

## M3. 预处理与特征编排
- 目标：支持面向分析师的可配置步骤链。
- 交付物：
  - 预处理步骤模型与保存接口
  - 特征步骤模型与保存接口
  - 样本预览与字段变化预览
  - 常用步骤组件与参数面板
- 验收标准：
  - 可保存并读取步骤链配置
  - 可按顺序执行步骤链
  - 预处理输出与特征输出可版本化

## M4. 训练与重训练
- 目标：完成监督、无监督训练任务和追加样本后重训练。
- 交付物：
  - 训练任务创建与任务状态查询
  - 内置算法选择
  - 模型版本记录
  - 基于新数据版本的重训练流程
- 验收标准：
  - 训练任务可异步执行
  - 同一项目可保留多个模型版本
  - 重训练不会覆盖历史版本

## M5. 评估与异常分析
- 目标：产出可视化评估和异常结果分析视图。
- 交付物：
  - 指标计算与图表数据接口
  - 异常列表与异常详情
  - 趋势分布与关键字段聚合
  - 模型版本对比视图
- 验收标准：
  - 有标签数据可生成完整评估结果
  - 无监督结果可显示异常分数和 Top-N 异常样本
  - 前端可查看异常详情与基础上下文

## M6. Ollama 集成与交付加固
- 目标：完成异常解释集成和本地交付完善。
- 交付物：
  - Ollama 连通性检查
  - 异常样本解释接口
  - AI 解释展示区
  - 启动、排障、配置文档
- 验收标准：
  - 未安装 Ollama 时系统可降级运行
  - 已配置 Ollama 时可返回异常解释结果
  - 本地部署步骤完整、可重复
